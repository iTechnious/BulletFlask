# BulletFlask
## Custom response code

| code  | meaning |
|-------|---------|
| 901   | Username already taken |
| 902   | Email already taken |
| 903   | Problem during login |
| 904   | Wrong password |
| 905   | User not found |
| 906   | Not permitted  |
